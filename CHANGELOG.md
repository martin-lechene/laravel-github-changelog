# Changelog

All notable changes to `laravel-github-changelog` will be documented in this file.

## 3.0.0 - 2026-01-08

### Breaking Changes
- **Minimum PHP version increased to 8.2** (PHP 7.4, 8.0, 8.1 are no longer supported)
- Removed support for Laravel 7 and 8

### Added
- Added support for Laravel 11
- Added support for Laravel 12
- Added support for PHP 8.3 and 8.4
- Added GitHub Actions CI workflow for modern automated testing

### Changed
- Updated PHPUnit to ^10.0 || ^11.0
- Updated Orchestra Testbench to ^7.0 || ^8.0 || ^9.0
- Modernized phpunit.xml configuration for PHPUnit 10/11 compatibility
- Updated Travis CI configuration to test PHP 8.2, 8.3, and 8.4

## 1.0.0 - 2020-10-16

-   initial release
